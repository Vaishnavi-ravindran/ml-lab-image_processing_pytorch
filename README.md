# Image Processing using PyTorch

This project demonstrates basic image processing techniques implemented using PyTorch.

## Features
- Histogram Equalization
- Image Thresholding
- Edge Detection (Sobel Operator)
- Data Augmentation
- Morphological Operations (Dilation & Erosion)

## Input
Grayscale image (image.jpg)

## Output
- Enhanced image (Histogram Equalization)
- Binary image (Thresholding)
- Edge detected image
- Augmented image
- Morphological transformations

## Tools Used
- Python
- PyTorch
- Torchvision
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Open notebook:
   image_processing_pytorch.ipynb

## Concepts Covered
- Image Processing
- Convolution operations
- Data augmentation
- Morphological transformations
